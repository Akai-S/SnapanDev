# Tugas Penilaian

## Deskripsi
Program ini digunakan untuk menentukan nilai huruf berdasarkan skor yang didapatkan oleh siswa.

## Kriteria Penilaian
- **A** : 100 – 90
- **B** : 89 – 80
- **C** : 79 – 70
- **D** : 69 – 60
- **E** : di bawah 60

## Cara Menggunakan
1. Masukkan nilai (0–100).
2. Program akan mengeluarkan nilai huruf sesuai rentang di atas.

## Contoh
- Input: `95` → Output: `A`
- Input: `85` → Output: `B`
- Input: `73` → Output: `C`
- Input: `62` → Output: `D`
- Input: `50` → Output: `E`
